# Sign-Up Form

A sign-up form project built as part of [The Odin Project](https://www.theodinproject.com/lessons/node-path-intermediate-html-and-css-sign-up-form), a free and open source curriculum for learning web development.

## Description
Ths project showcases a sign-up form interface featuring a branded left sidebar and a comprehensive formlayout on the right.

## Features
* Relevant input verification using the following HTML input element attributes:
    * required
    * minlength
* Custom styling using CSS :user-valid and :user-invalid pseudo-classes
* Inter font family

## Built With
* HTML
* CSS
* Flexbox

## Notes
* This is a static HTML/CSS project (form submission is not functional)
